# job-track-mobile-app

A modern cross-platform mobile application built using React Native and Expo ecosystem. The app is optimized for Android devices and uses Hermes for improved JavaScript performance.

---

## Tech Stack

### Frontend
- React Native
- Expo
- JavaScript / TypeScript

### Runtime Engine
- Hermes Engine

### Native Android
- Kotlin
- AndroidX

### UI & Animations
- React Native Reanimated
- React Native Gesture Handler
- React Native Screens
- React Native SVG

### Networking
- OkHttp
- REST APIs

---

## Architecture Overview

Frontend:
React Native + Expo

Runtime:
Hermes Engine

Native Layer:
Kotlin + AndroidX

Networking:
OkHttp + REST APIs

UI Layer:
Reanimated + Gesture Handler + SVG

---

## Features

- Cross-platform mobile architecture
- Smooth animations and gesture handling
- Optimized runtime performance using Hermes
- Modern Android support with AndroidX
- Scalable modular structure
- REST API integration

## Build System

- Android APK packaging

- Expo native module integration
